	                   Readme for the POMC Package
	 		       version May 25, 2017
------------------------------------------------------------------------------------------

The package includes the JAVA code of the POMC algorithm [1] for maximizing monotone set functions with monotone cost constraints.

[1] Chao Qian, Jing-Cheng Shi, Yang Yu and Ke Tang. On subset selection with general cost constraints. In: Proceedings of the 26th International Joint Conference on Artificial Intelligence (IJCAI'17), Melbourne, Australia, 2017.

In the 'Example.java' file, you will find an example of using this code for influence maximization with routing constraints.
